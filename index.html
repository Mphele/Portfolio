afya conversational medical assistant python · fastapi 2026 →

Afya is a conversational medical assistant engineered to bridge the healthcare accessibility gap in Africa. Built for communities where seeing a doctor isn't always an option, the platform provides instant symptom triage and actionable medical guidance in native languages.

To keep the user experience deliberately simple and fast, the backend architecture handles immense complexity under the hood. The FastAPI server utilizes thread pooling to orchestrate concurrent dynamic translation, AI severity-parsing, and geolocation routing to the nearest physical clinics without blocking the event loop. To guarantee accessibility and empathy in critical moments, the system features a custom, fault-tolerant text-to-speech pipeline. This pipeline dynamically manages language fallbacks and generates synchronized visemes, driving a 3D frontend avatar that lip-syncs the streaming auditory guidance in real-time.

sieve recruiter-first ATS typescript · node.js · express · mongodb 2026 →

The Problem: Most ATS platforms optimize for the hiring manager using rigid algorithms that strip control from the recruiters actually reading the resumes, inevitably leading to automated false positives.

The Solution: Sieve is an AI-assisted screening tool designed to empower recruiter intuition rather than replace it. The system actively parses candidate data and provides comparative scoring, while maintaining a strict "human-over-algorithm" workflow that allows recruiters to manually override rankings and adjust scoring criteria.

The Architecture: A robust RESTful API engineered with TypeScript, Node.js, Express, and MongoDB. The backend deeply integrates large language models to dynamically parse PDF resumes into structured JSON, generate role-specific evaluation rubrics from raw job descriptions, and stream real-time candidate Q&A directly to the client via Server-Sent Events (SSE).
